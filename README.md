Example Node Server
=============

Setup:
  * Install nodemon:
    * `npm install -g nodemon`
  * Install dependencies:
    * `npm install` (This installs all the dependencies in package.json)

To run this server:

  * `nodemon server.js`
  * Visit http://localhost:5000/

To view the server in action:

  * Visit http://frozen-plains-7894.herokuapp.com/
